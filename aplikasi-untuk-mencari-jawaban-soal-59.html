Pada عَلَيْهِنَّ terdapat dua hukum tajwid yaitu mad layyin ( karena ya sukun sebelumnya huruf berharkat fathah) dan ghunnah

Pada مِنْ جَلَابِيبِهِنَّ terdapat hukum tajwid yaitu ikhfa ( karena nun mati bertemu dengan huruf jim), mad asli dan ghunnah

Pada يُؤْذَيْنَ terdapat hukum tajwid mad layyin

37. carilah hukum bacaan Qs.An-nisa:59.buat sejelas mungkin​


Hukum tajwid dari masing-masing kode tersebut :

A2 => Idgham Bighunnah.A5 => Ikhfa Haqiqi.B1 => Izhar Syafawi.D1 => Alif Lam Qamariah.D2 => Alif Lam Syamsiyah.E1 => Tafkhim.E2 => Tarqiq.F1 => Mad Thobi'i.F3 => Mad Jaiz Munfasil.F5 => Mad Iwad.

PEMBAHASAN

➡ Berikut penjelasan dari masing-masing kode tajwid tersebut :

1. A2 ( Idgham Bighunnah )

Pada lafaz خَيْرٌ وَّ ( karena ada tanwin diikuti huruf waw ).

2. A5 ( Ikhfa Haqiqi )

Pada lafaz مِنْكُمْۚ ( karena ada nun mati bertemu dengan kaf ).Pada lafaz شَيْءٍ فَرُ ( karena ada tanwin diikuti huruf FA ).

3. B1 ( Izhar Syafawi )

Pada lafaz زَعْتُمْ فِيْ ( karena ada mim mati bertemu dengan fa ).

4. D1 ( Alif Lam Qamariah )

Pada lafaz الْاَمْرِ ( karena ada Alif lam diikuti huruf Alif ).Pada lafaz الْاٰخِرِۗ ( karena ada Alif lam diikuti huruf Alif ).Pada lafaz وَالْيَوْمِ ( karena ada Alif lam diikuti huruf ya ).

5. D2 ( Alif Lam Syamsiyah )

Pada lafaz الرَّسُوْلَ ( karena ada Alif lam diikuti huruf Ra ).

6. E1 ( Tafkhim )

Pada lafaz اَطِيْعُوا اللّٰهَ ( karena ada lafaz Allah didahului huruf berharoqah dhommah ).Pada lafaz اِلَى اللّٰهِ ( karena ada lafaz Allah didahului huruf berharoqah Fatah ).

7. E2 ( Tarqiq )

Pada lafaz بِاللّٰهِ ( karena lafaz Allah didahului huruf berharoqah kasroh ).

8. F1 ( Mad Thobi'i )

Pada lafaz الَّذِيْنَ ( karena ada kasroh diikuti huruf ya sukun ).Pada lafaz تُؤْمِنُوْنَ ( karena ada dhommah diikuti huruf waw sukun ).

9. F3 ( Mad Jaiz Munfasil )

Pada lafaz يَا أَيُّهَا ( karena ada mad Thobi'i bertemu dengan Hamzah tetapi tidak dalam satu kata ).Pada lafaz آمَنُوا أَطِيعُ ( karena ada mad Thobi'i bertemu dengan Hamzah tetapi tidak dalam satu kata ).

10. F5 ( Mad Iwad )

Pada lafaz وِيْلًا ( karena ada lam berharakat fatahatain yang diwaqofkan).
