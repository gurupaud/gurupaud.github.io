= (2021 × 2021) - (2022 × 2020) + 2

= 4.084.441 - 4.084.440 + 2

= 1 + 2

= 3

Q = 2018 × 2020 - 2019²

= 4.076.360 - (2019 × 2019)

= 4.076.360 - 4.076.361

= -1
