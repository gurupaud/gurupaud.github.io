4. matematika hal 93 kelas 8 kurikulum 2013


Lengkapi tabel berikut halaman 93 Buku Matematika kelas 8 smp semester 2

Pada bab ini kita masih membahas soal Lingkaran, yaitu tentang sudut pusat, panjang busur dan Luas Juring

Sudut pusat adalah sudut yang dibentuk oleh dua jari-jari yang berpotongan pada pusat lingkaran

Busur adalah garis lengkung yang menghubungkan dua titik sembarang pada lengkungan lingkaran

Juring adalah daerah di dalam lingkaran yang dibatasi oleh dua jari-jari lingkaran dan busur lingkaran dihadapan sudut pusat yang dibentuk oleh kedua jari-jarinya

Rumus :

Luas Juring = [tex]\frac{\theta}{360}[/tex]L

Busur = [tex]\frac{\theta}{360}[/tex]K

Dimana :

[tex]\theta[/tex] = sudut juring dalam derajat

K = keliling lingkaran (K = 2πr )

L = Luas lingkaran (L = πr² )

Pembahasan :

1. a. Diketahui :

[tex]\theta[/tex] = 90°

r = 7cm

π = [tex]\frac{22}{7}[/tex]

Ditanya : Panjang Busur ?

Dijawab :

Keliling = 2πr

= 2 x [tex]\frac{22}{7}[/tex] x 7

= 44 cm

Busur = [tex]\frac{\theta}{360}[/tex]K

= [tex]\frac{90}{360}[/tex] x 44

= 11 cm


b. Diketahui :

[tex]\theta[/tex] = 60°

r = 21cm

π = [tex]\frac{22}{7}[/tex]

Ditanya : Panjang Busur ?

Dijawab :

Keliling = 2πr

= 2 x [tex]\frac{22}{7}[/tex] x 21

= 132 cm

Busur = [tex]\frac{\theta}{360}[/tex]K

= [tex]\frac{60}{360}[/tex] x 132

= 22 cm


c. Diketahui :

[tex]\theta[/tex] = 120°

Panjang Busur = 88 cm

π = [tex]\frac{22}{7}[/tex]

Ditanya : r ?

Dijawab :

Busur = [tex]\frac{\theta}{360}[/tex]K

88 = [tex]\frac{120}{360}[/tex] x K

88 = [tex]\frac{1}{3}[/tex] x K

K = 264 cm

Keliling = 2πr

264 = 2 x [tex]\frac{22}{7}[/tex] x r

264 = [tex]\frac{44}{7}[/tex] x r

r = 42 cm


d. Diketahui :

r = 100 cm

Panjang Busur = 31,4 cm

π = 3,14

Ditanya : [tex]\theta[/tex] = ?

Dijawab :

Keliling = 2πr

= 2 x 3,14 x 100

= 628 cm

Busur = [tex]\frac{\theta}{360}[/tex]K

314 = [tex]\frac{\theta}{360}[/tex] x 628

[tex]\frac{\theta}{360}[/tex] = [tex]\frac{314}{628}[/tex]

[tex]\frac{\theta}{360}[/tex] = [tex]\frac{1}{2}[/tex]

[tex]\theta[/tex] = [tex]\frac{360}{2}[/tex]

[tex]\theta[/tex] = 180°


e. Diketahui :

[tex]\theta[/tex] = 72°

Panjang Busur = 1.256 cm

π = 3,14

Ditanya : r ?

Dijawab :

Busur = [tex]\frac{\theta}{360}[/tex]K

1.256 = [tex]\frac{72}{360}[/tex] x K

1.256 = [tex]\frac{1}{5}[/tex] x K

K = 6.280 cm

Keliling = 2πr

6.280 = 2 x 3,14 x r

6.280 = 6,28r

r = 1000 cm


2. a. Diketahui :

[tex]\theta[/tex] = 100°

r = 6cm

π = 3,14

Ditanya : Luas Juring ?

Dijawab :

L = πr²

L = 3,14 x 6 x 6

L = 113,04 cm²

Luas Juring = [tex]\frac{\theta}{360}[/tex]L

= [tex]\frac{100}{360}[/tex] x 113,04

= 31,4 cm²


b. Diketahui :

[tex]\theta[/tex] = 25°

Luas juring = 31,4 cm²

π = 3,14

Ditanya : r ?

Dijawab :

Luas Juring = [tex]\frac{\theta}{360}[/tex]L

31,4 = [tex]\frac{25}{360}[/tex] x L

31,4 = [tex]\frac{5}{72}[/tex] x L

L = 452,16 cm²

L = πr²

452,16 = 3,14 x r²

r² = 144

r = 12 cm


c. Diketahui :

r = 90 cm

Luas juring = 8.478 cm²

π = 3,14

Ditanya : [tex]\theta[/tex] = ?

Dijawab :

L = πr²

L = 3,14 x 90 x 90

L = 25.434 cm²

Luas Juring = [tex]\frac{\theta}{360}[/tex]L

8.478 = [tex]\frac{\theta}{360}[/tex] x 25.434

[tex]\frac{\theta}{360}[/tex] = [tex]\frac{8.478}{25.434}[/tex]

[tex]\theta[/tex] = 120°

15. Ayo kita berlatih 3.3 buku matematika kelas 7 kurikulum 2013 hal 222


Ayo kita berlatih 3.3 buku matematika kelas 7 kurikulum 2013 hal 222JawabanPendahuluan

Pada "Ayo kita berlatih 3.3" halaman 222 terdapat 10 soal esay. Saya akan menjelaskan hanya 5 soal saja yaitu dari no 1 sampai 5. Untuk 5 soal tersebut bisa lihat pada lampiran.

Pembahasan

1.  Tentukan hasil kali bentuk aljabar.

a.  10 (2y - 10) = (10 × 2y) + (10 × -10)

                       = 20y - 100

b.  (x + 5) (5x - 1) = (x × 5x) + (x × -1) + (5 × 5x) + (5 × -1)

                           = 5x² - x + 25x - 5

                           = 5x² + 24x - 5

c.  (7 - 2x) (2x - 7) = (7 × 2x) + (7 × -7) + (-2x × 2x) + (-2x × -7)

                             = 14x - 49 - 4x² + 14x

                             = 28x - 49 - 4x²

2.  Tentukan nilai r pada persamaan bentuk aljabar (2x + 3y)(px + qy) = rx² + 23xy + 12y²

                                       (2x + 3y)(px + qy) = rx² + 23xy +12y²

[2x × px] + [2x × qy] + [3y × px] + [3y × qy] = rx² + 23xy +12y²

                         2px² + (2q + 3p) xy + 3qy² = rx² + 23xy +12y²

2px² = rx²    (sama-sama dicoret x²)

  2p = r

(2q + 3p) xy = 23 xy     (sama-sama dicoret xy)

      2q + 3p = 23

3qy² = 12y²    ((sama-sama dicoret y²)

  3q = 12

 q = 12/3

    q = 4

 2q + 3p = 23 

2(4) + 3p = 23 

   8 + 3p = 23 

         3p = 23 - 8

         3p = 15

           p = 5

  2p = r

2(5) = r 

    r = 10

3.  Nyatakan luas bangun datar berikut dalam bentuk aljabar.

    a.  Luas persegi = sisi × sisi

                               = (a + a) × (a + a)

                              = 2a × 2a

                             = 4a²

    b.  Luas persegi panjang = p × l

                                              = 3a × 3b

                                              = 9ab

    c.  Luas = (2s + t) × 3s

                 = 6s² + 3st

4.  Tentukan dua bentuk aljabar yang bila dikalikan hasilnya adalah

   a.  102 × 98 = (100 + 2)(100 - 2)

                       = 100² - 2²

                       = 10.000 - 4

                       = 9.996

   b.  1003 × 97 = (1000 + 3)(100 - 3)

                         = 100.000 - 3000 + 300 - 9

                         = 97.291

   c.  205² = (200 + 5)²

                 = 40.000 + 2.000 + 25

                 = 42.025


   d.  389² = (400 - 11)²

                 = 400² - (400) (11) (2) + 11²

                 = 160.000 - 8.800 + 121

                 = 151321

5.  Bagaimana cara menentukan perpangkatan bentuk aljabar berikut ? jelaskan !

Untuk menjawab soal a kita bisa menggunakan segitiga pascal.

a. (a + b)⁵ = 1(a)⁵ (b)⁰ + 5 (a)⁴ (b)¹ + 10 (a)³ (b)² + 10 (a)² (b)³ + 5 (a)¹ (b)⁴ + 1(a)⁰ (b)⁵

               = a⁵ + 5 a⁴ b + 10 a³ b² + 10 a² b³ + 5 a b⁴ + b⁵

untuk b sampai e kita bisa menggunakan perkalian aljabar.

b.  (a + b + c)² = (a + b + c) (a + b + c)

                       = a² + ab + ac + ab + b² + bc + ac + bc + c²

                       = a² + b² + c² + 2ab + 2ac + 2bc

                       = a² + b² + c² + 2 (ab + ac + bc)

c.  (a + b - c)² = (a + b - c) (a + b - c)

                     = a² + ab - ac + ab + b² - bc - ac - bc + c²

                     = a² + b² + c² + 2ab - 2ac - 2bc

                     = a² + b² + c² + 2 (ab - ac - bc)

d.  (a - b + c)² = (a - b + c) (a - b + c)

                     = a² - ab + ac - ab + b² - bc + ac - bc + c²

                     = a² + b² + c² - 2ab + 2ac - 2bc

                     = a² + b² + c² - 2 (ab - ac + bc)

e.  (a - b - c)² = (a - b - c) (a - b - c)

                     = a² - ab - ac - ab + b² + bc - ac + bc + c²

                     = a² + b² + c² - 2ab - 2ac + 2bc

                     = a² + b² + c² - 2 (ab + ac - bc)

Kesimpulan

Soal 7.

Berilah nama untuk setiap titik koordinat yang hilang dalam setiap gambar berikut

a. Persegi ABCD

A = (0,0), B = (a,0), C = (a,a), D = (0,a)

karena persegi, maka memiliki sisi-sisi yang samam dimana OB = BC = DC = DO = a

b. Persegipanjang EFGH

E = (0,0), F = (a,0), G = (a,b), H = (0,b)

karena persegi panjang, maka memiliki sisi-sisi dimana HG = EF = a, dan HE = GF = b

Soal 8.

Segitiga siku-siku QRS digambar dalam bidang koordinat sebagai berikut

a. Jika QR = 6 stuan dan QS = 4 satuan, tentukan koordinat titik-titik segitiga QRS

Perhatikan gambar terlampir, karena QR = 6 satuan, maka titik Q terletak saat x dan y nya nol, sedangkan titik R terletak saat x = 6 dan y = 0. Lalu pada garis SR, maka titik S terletak ketika x = 0 dan y = 4

Q (0,0), R(6,0), S(0,4)

b. Jika QR = A dan QS = B, tentukan koordinat Q, R , dan S

Perhatikan gambar terlampir, karena soal a sudah diketahui Q (0,0), R(6,0), S(0,4) , maka 6 adalah a dan 4 adalah b:

Q (0,0), R(a,0), S(0,b)
