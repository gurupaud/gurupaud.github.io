a.)15 km + 25 hm = ... dam

= (15 x 100) + (25 x 10) = .... dam

= 1.500 + 250 = 17.50 dam

b.) 1,75 km - 20 dam = ...m

= (1,75 x 1.000) + (20 x 10) = ....m

= 1.750+ 200 = 1950m

c.) 45 dm + 750 mm = ... cm

= (45 x 10) + (750 / 10) =... cm

= 450 + 75 = 525 cm

d.) 5 km - 12 hm - 6 dam = ... dam

= (5 x 100) + (12 x 10) - 6 =... dm

= 500 + 120 - 6 = 614dm

f.) 20 m + 120 dm - 270 cm = ... dm

= (20 x 10) + 120 + (270 / 10) = .... dm

= 200 + 120 + 27= 293 dm

e.) 2.500 km - 2.500 dam = .... km

= 2.500 - (2.500 / 100) = ..... km

= 2.500 - 25 = 2.475 km

g.) 1.500 mm + 25 m = ...mm

= 1.500 + (25 x 1.000) = ...mm

= 1.500 + 25.000 = 26.500 mm

h.) 6 dam - 150 dm = ... mm

= (6 x 10.000) - (15 x 100) = ... mm

= 60.000 - 1.500 = 58.500 mm

i.) 75 km - 12 hm + 150 m = ... dam

= (75 x 100) - (12 x 10)+(150 / 10) = ... dm

= 7.500 - 120 + 15 = 7.395 dm

j.) 6,2 km - 22 hm - 3.750 m = ...m

= (6,2 x 1000) - (22 x 100) - 3.750 = ... m

= 6.200 - 2.200 - 3.750 = 250 m
