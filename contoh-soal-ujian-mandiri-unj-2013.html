10. SOAL UJIAN MASUK UM UGM 2013 SBMPTN 2016


Jawaban: C

Soal ini lebih mudah diselesaikan dengan mensubstitusikan pilihan jawaban ke soal dan mengeliminasi jawaban yang salah:

Ambil x = 2, maka diperoleh:

3√(2 - 2)/ 2 - 1 < 2 ⇒ 0 < 2 (BENAR)

Pilihan jawaban yang tidak memuat x = 2 salah sehingga pilihan jawaban A, B, E salah.

Kemudian ambil x = 0, maka diperoleh:

3√(2 - 0)/ 0 - 1 < 2 ⇒ 3√2/-1 < 2 (BENAR)

Pilihan jawaban yang tidak memuat x = 0 salah sehingga pilihan jawaban D salah.
Jadi, jawaban yang benar adalah C.

24. tolong bantu ya soal ujian mandiri


ax² + 5x - 12 = 0

Salah satu akar persamaannya adalah 2, jadi:


(ax + .......) (x - 2) = 0 ⇒ salah satu akar persamaannya adalah 2 ( yaitu x-2)


Isian titik-titik tersebut adalah:


...... di kali -2 = -12 ⇒ jadi titik-titik tersebut adalah 6.


Jadi: (ax + 6) (x - 2) = 0


Untuk mencari nilai a adalah:


(ax dikali -2) + (6 dikali x) = 5x

-2ax + 6x = 5x

-2a + 6 = 5

-2a = -1

a = 1/2



Jadi ⇒ (1/2x + 6) (x - 2) = 0


Akar persamaan yang lainnya adalah:


(1/2x + 6) = 0

1/2x = -6

x = -12


Jawaban A

Jawaban: B

Karena (b + c, b, c) memenuhi persamaan, maka:
3(b + c) - b + 2c = -1
3b + 3c - b + 2c = -1
2b + 5c = -1........................Pers.1

-2(b + c) + b + 3c = -3
-2b - 2c + b + 3c = -3
-b + c = -3...........................Pers.2

2b + 5c = -1
-2b + 2c = -6
_____________ +

7c = -7
c = -1

-b + (-1) = -3
b = 2

Jadi, b + c = 2 + (-1) = 1
