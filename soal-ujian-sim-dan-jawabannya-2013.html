2. Peluang Reno untuk lulus ujian SIM adalah 70%. Sedangkan peluang Seto untuk lulus ujian SIM adalah 80%. Peluang keduanya bernasib sama saat ujian SIM adalah ....​


Penjelasan dengan langkah-langkah:

Peluang Reno Lulus => P (Rl) = 70% = 0,7

Peluang Seto Lulus => P (Sl) = 80% = 0,8

Peluang keduanya bernasib sama, berarti ketika keduanya lulus dan keduanya tidak lulus

• Keduanya Lulus

P (Rl Sl) = P(Rl) × P(Sl) = 0,7 × 0,8 = 0,56 = 0,56 × 100% = 56%

• Keduanya Tidak Lulus

Peluang Reno Tidak Lulus => P (Rtl) = 100% - 70% = 30% = 0,3

Peluang Seto Tidak Lulus => P (Stl) = 100% - 80% = 20% = 0,2

P (Rtl Stl) = P(Rtl) × P(Stl) = 0,3 × 0,2 = 0,06 = 0,06 × 100% = 6%

4. SOAL UJIAN MASUK UM UGM 2013 SBMPTN 2016


Jawaban: C

Soal ini lebih mudah diselesaikan dengan mensubstitusikan pilihan jawaban ke soal dan mengeliminasi jawaban yang salah:

Ambil x = 2, maka diperoleh:

3√(2 - 2)/ 2 - 1 < 2 ⇒ 0 < 2 (BENAR)

Pilihan jawaban yang tidak memuat x = 2 salah sehingga pilihan jawaban A, B, E salah.

Kemudian ambil x = 0, maka diperoleh:

3√(2 - 0)/ 0 - 1 < 2 ⇒ 3√2/-1 < 2 (BENAR)

Pilihan jawaban yang tidak memuat x = 0 salah sehingga pilihan jawaban D salah.
Jadi, jawaban yang benar adalah C.

X = [75nA + 70(nA + 6)] / nA + nB
72 = [75nA + 70(nA + 6)] / nA + (nA + 6)
72 = (145nA + 420)/ 2nA + 6
144nA + 432 = 145nA + 420

40. Probabilitas seseorang lulus ujian sim adalah 0,6. Jika seseorang mencoba terus menerus sampai lulus ujian sim. a.berapa probabilitasnya dia akan lulus pada ujian ke empat? b.berapa probabilitasnya dja akan lulus sebelum mengikuti ujian keempat?


kemungkinan lulus [tex]p = 0,6[/tex]

kemungkinan tidak lulus [tex]q = 1-p = 0,4[/tex]

maka

lulus pada ujian ke [tex]x[/tex] berarti mengalami tidak lulus sebanyak [tex]x-1[/tex] kali, sehingga kemungkinan lulus pada ujian ke [tex]x[/tex] dapat dinyatakan dengan

[tex]P(X=x) = q^{x-1}p[/tex]

maka

a)

[tex]P(X=4) = q^{4-1}p = (0,4)^3(0,6)\\\boxed{P(X=4) = 0,0384}[/tex]

b)

[tex]P(X<4) = P(3) + P(2) + P(1)\\= q^2p  + qp + p\\= (0,4)^2(0,6) + (0,4)(0,6) + 0,6\\\boxed{P(X<4) = 0,936}[/tex]
