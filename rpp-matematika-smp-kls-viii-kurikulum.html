9. Analisis rpp kurikulum karakter dan kurikulum merdeka. Carilah perbedaan dan persamaan yang terdapat dalam rpp kurikulum karakter dan RPP kurikulum merdeka


Penjelasan:

Perbedaan

1. Fokus:

- Kurikulum Karakter: Kurikulum karakter bertujuan untuk mengembangkan karakter positif dalam siswa, seperti moral, etika, kejujuran, tanggung jawab, disiplin, kerja keras, dan sikap positif lainnya.

- Kurikulum Merdeka: Kurikulum Merdeka memiliki fokus pada pengembangan kemandirian dan pemikiran kritis siswa, mengajarkan keterampilan hidup, seperti berpikir, bertanya, berkomunikasi, berkolaborasi, dan mengelola diri sendiri.

2. Pendekatan:

- Kurikulum Karakter: Kurikulum karakter sering kali mengintegrasikan nilai-nilai karakter ke dalam semua mata pelajaran dan kegiatan sekolah. Pendekatan yang digunakan melibatkan pengajaran langsung dengan menggunakan metode dan teknik seperti cerita, permainan peran, diskusi, dan penilaian karakter.

- Kurikulum Merdeka: Kurikulum Merdeka mendorong pendekatan pembelajaran yang aktif dan kolaboratif. Siswa diberikan kebebasan untuk mengkaji materi secara mandiri dan memilih cara belajar yang paling sesuai untuk mereka. Guru bertindak sebagai fasilitator yang membantu siswa mengembangkan keterampilan pemecahan masalah dan pemikiran kritis.

3. Tujuan:

- Kurikulum Karakter: Tujuan dari Kurikulum Karakter adalah mengembangkan nilai-nilai karakter yang kuat pada siswa, sehingga mereka menjadi pribadi yang bertanggung jawab, berintegritas, dan memiliki empati terhadap orang lain.

- Kurikulum Merdeka: Tujuan dari Kurikulum Merdeka adalah mengembangkan siswa yang mandiri, aktif, dan kreatif, yang mampu mengambil tanggung jawab atas pembelajaran mereka sendiri. Mereka diharapkan dapat menghadapi tantangan dunia nyata dengan pemikiran kritis dan kemampuan beradaptasi.

Persamaan antara RPP Kurikulum Karakter dan RPP Kurikulum Merdeka adalah:

1. Menekankan pengembangan keterampilan hidup siswa.

2. Mengajarkan nilai-nilai positif dan moralitas.

3. Memprioritaskan pengembangan kepribadian siswa.

4. Mengikuti kurikulum yang ditetapkan secara nasional atau oleh lembaga pendidikan terkait.

Sifat-Sifat Bilangan Berpangkat
a. Sifat Perkalian Bilangan Berpangkat
Jika m, n adalah bilangan bulat positif dan a bilangan bulat maka am x an = am + n.
Contoh Soal :
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa 36 x 33 = 39!
Jawab:
36 x 33 = ( 3 x 3 x 3 x 3 x 3 x 3) x (3 x 3 x 3)
6 faktor 3 faktor
= (3 x 3 x 3 x 3 x 3 x 3 x 3 x 3 x 3)
(6 + 3) faktor
= 26 + 3
= 29

b. Sifat Pembagian Bilangan Berpangkat
Jika m, n bilangan bulat positif dan m > n dengan a bilangan bulat dan a bukan 0, maka am : an = am - n.
Contoh Soal :
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa 36 : 33 = 33!
Jawab :
36 : 33 = 36
33
= (3 x 3 x 3 x 3 x 3 x 3)
3 x 3 x 3
= (3 x 3 x 3)
(6 - 3) faktor
= 36 - 3
= 33

c. Sifat Perpangkatan Bilangan Berpangkat
Jika a, m, dan n bilangan bulat maka (am)n = am x n.
Contoh Soal:
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa (43)2 = 46.
Jawab:
(43)2 = (43) x (43)
= (4 x 4 x 4) x (4 x 4 x 4)
3 faktor 3 faktor
= 4 x 4 x 4 x 4 x 4 x 4
(3 x 2) faktor
= 43 x 2 = 46

d. Sifat Perpangkatan pada Perkalian
Jika a, b bilangan bulat dan m bilangan bulat positifmaka (a x b)m = am x bm.
Contoh Soal=
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa (32)3 = 33 x 23.
Jawab:
(32)3 = (3 x 2) x (3 x 2) x (3 x 2)
= (3 x 3 x 3) x (2 x 2 x 2)
3 faktor 3 faktor
= 33 x 23

e. Sifat Perpangkatan pada Pembagian.
Jika a, b bilangan bulat (b bukan 0) dan m bilangan bulat positif, maka (a : b)m = am : bm.
Contoh soal:
Dengan cara menuliskan dalam bentuk pembagian, tunjukkan bahwa (5 : 3)4 = 54 : 34.
Jawab:
(5 : 3)4 = (5/3)4
= (5/3) x (5/3) x (5/3) X (5/3)
= (5 x 5 x 5 x 5) = 54 = 54 : 53
(3 x 3 x 3 x 3) 34
