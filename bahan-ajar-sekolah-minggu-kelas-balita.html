Persamaan linier dua variabel merupakan persamaan al-jabar dimana setiap variabelnya berderajat satu.

Penyelesaian persamaan linier dua variabel dapat dengan beberapa metode, diantaranya, metode eliminasi, subsitusi dan gabungan dari kedua metode tersebut.

Metode eliminasi merupakan suatu metode yang digunakan untuk memecahkan atau mencari himpunan penyelesaian suatu sistem persamaan linear dua variabel dengan cara menghilangkan (mengeliminasi) salah satu variabelnya.

Metode subtitusi adalah cara menyelesaikan persamaan dengan memasukkan salah satu persamaan ke dalam persamaan yang lain.

Pembahasan

Untuk menyelesaian soal diatas kita gunakan metode subsitusi

Jawab:

Penjelasan dengan langkah-langkah:

Diketahui:

b + d = 540

b = 540 - d .......................................... persamaan 1

2000b + 3000d = 1.260.000 ......... persamaan 2

Langkah pertama, subsitusikan persamaan 1 ke persamaan 2

2000b + 3000d = 1.260.000

2000(540 - d) + 3000d = 1.260.000

1.080.000 - 2000d + 3000d = 1.260.000

1.080.000 + 1000d = 1.260.000

1000d = 1.260.000 - 1.080.000

1000d = 180.000

d = 180.000 : 1000

d = 180 tiket

Langkah kedua, subsitusikan nilai d ke persamaan 1

b = 540 - d

b = 540 - 180

b = 360 tiket

Jadi banyak tiket balita yang terjual ada 360 tiket dan banyak tiket dewasa yang terjual ada 180 tiket

32. Tiket masuk ruang pameran untuk balita Rp.2.000,00 dan untuk dewasa Rp.3.000,00. Pada hari Minggu, 540 tiket terjual dengan penjualan Rp1.260.000,00. Berapa banyak tiket masuk untuk balita yang dijual?​


Jawaban:

asumsikan :

tiket balita = a

tiket dewasa = b

maka :

a + b = 540 (pers. 1)

2000a + 3000b = 1.260.000 (pers. 2)

Substitusi pers. 1 menjadi :

a = 540 - b (pers. 3)

Substitusi pers. 3 ke dalam pers. 2

2000a + 3000b = 1.260.000

2000 (540 - b) + 3000b = 1.260.000

1.080.000 - 2000b + 3000b = 1.260.000

-2000b + 3000b = 1.260.000 - 1.080.000

1000b = 180.000

b = 180.000/1000

b = 180

Substitusi nilai b ke pers. 1 untuk mencari nilai a

a + b = 540

a + 180 = 540

a = 540 - 180

a = 360

maka tiket balita yang dijual sebanyak 360 tiket
