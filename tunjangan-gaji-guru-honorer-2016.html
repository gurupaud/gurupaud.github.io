5. tunjangan gaji tentara


1.gaji pokok.
gaji pokok tni saat ini masih mengaju pada PP Nomor 31 tahun 2015.sama kayak gaji PNS,gaji pokok tni juga ditetapkan berdasarkan golongan dan masa kerja. tentu saja besarnya berbeda..

1.golongan:tamtama
Rp terendah:1.565.200
Rp tertinggi:2.819.500

2.golongan:bintara
Rp terendah:2.003.300
Rp tertinggi:3.839.800

3.golongan:perwira pertama.
Rp terendah:2.604.400
Rp tertinggi:4.551.700

4.golongan:perwira menengah-tinggi
Rp terendah:2.856.400
Rp tertinggi:5.464.100

jadi hingga saat ini gaji pokok tni paling kecil sebesar Rp 1.56jt dan paling besar senilai Rp.5.6jt.

2.tunjangan istri/suami tni.
jika sudah menikah, anggota tni berhak mendapatkan tunjangan istri/suami sebesar 10% dari gaji pokok.

3.tunjangan anak.
bagi anggota tni yang memiliki anak berhak mendapat tunjangan anak. tunjangan ini diberikan maksimal utk 2 anak masing2 sebesar 2% dari gaji pokok.

4.tunjangan beras.
berdasarkan peraturan dirjen perbendaharaan no 3/2015,harga pembelian beras yg dikeluarkan Negara utk tni sebesar Rp. 8.047 per kg.

5.tunjangan jabatan.
bagi anggota tni yg menduduki jabatan struktural,ia berhak menerima tunjangan jabatan.besaran tunjangan tni yg terendah rp. 360 ribu per bulan dan tertinggi sebesar rp. 5. 5jt.

6.uang lauk pauk tni
pada tahun 2018,tunjangan lauk pauk naik dari Rp.50rb menjadi Rp.60rb per hari.

7.tunjangan oprasi pengamanan
bagi anggota yg bertugas di pulau/wilayah yang kecil berhak mendapatkan tunjangan oprasi pengamanan.besarnya bervariasi tergantung beban kerja yg dimiliki.

8.tunjangan kinerja.
tunjangan kinerja alias tukin inilah yg mendapatkan kenaikan hingga 70% per bulan. walau begitu, besarnya masih perlu peraturan presiden yg baru

int main()

{

   double gaji_pokok, hasil_tunjangan, hasil_pajak;

   double tunjangan = 0.2;

   double pajak = 0.15;

   

   printf("Masukkan gaji anda : ");

   scanf("%lf",&gaji_pokok);

   printf("Gaji anda = %lf \n", gaji_pokok);

   hasil_tunjangan = gaji_pokok * tunjangan;

   printf("Tunjangan : %lf \n", hasil_tunjangan);

   hasil_pajak = (hasil_tunjangan + gaji_pokok) * pajak;

   printf("Pajak : %lf",hasil_pajak);

   return 0;

}

Untuk mengisi kolom ini, cukup dengan memasukkan nilai gaji pokok pada masing-masing baris.

Kolom Tunjangan Istri:

Jika tunjangan istri diberikan sebesar 10% dari gaji pokok, maka rumusnya sebagai berikut: =Gaji_Pokok*10%

Kolom Tunjangan Anak:

Jika tunjangan anak diberikan sebesar Rp. 500.000,- per anak, maka rumusnya sebagai berikut: =500000*Jumlah_Anak

Kolom Gaji Kotor:

Gaji kotor adalah jumlah dari gaji pokok, tunjangan istri, dan tunjangan anak. Maka rumusnya sebagai berikut: =Gaji_Pokok+Tunjangan_Istri+Tunjangan_Anak

Kolom Pajak:

Jika pajak dikenakan sebesar 5% dari gaji kotor, maka rumusnya sebagai berikut: =Gaji_Kotor*5%

Kolom Gaji Bersih:

Gaji bersih adalah hasil dari gaji kotor dikurangi dengan pajak. Maka rumusnya sebagai berikut: =Gaji_Kotor-Pajak

22. Buatlah program untuk menghitung gaji pegawai Input : nama, golongan Proses : · Jika golongan A maka gaji pokok =Rp 1.500.000, tunjangan =5% * gaji pokok .jika gol.B maka gaji pokok = Rp.1.700.000,tunjangan = 10%* gaji pokok .jika gol.C maka gaji pokok = Rp.2.000.000, tunjangan = 15%*gaji pokok Output : Nama,gaji pokok,tunjangan serta gaji bersih Petunjuk : gunakan array dan petualangan


program untuk menghitung gaji pegawai, program ditulis dalam bahasa C

#include <stdio.h>

int main()

{

    char nama[50];

    char golongan;

    int gaji[3] = { 1500000, 1700000, 2000000 };

    int persentunjangan[3] = { 5, 10, 15 };

    int gajibersih, tunjangan;

    printf("Masukan nama Anda : ");

    scanf("%s", nama);

    printf("Masukan golongan (A, B, atau C) : ");

    scanf("%c", &golongan);

    golongan = golongan - 'A';

    tunjangan = gaji[ golongan ] * persentunjangan[ golongan ] / 100;

    gajibersih = gaji[ golongan ] + tunjangan;

    printf("Nama : %s\n", nama);

    printf("Gaji pokok : %i\n", gaji[ golongan ]);

    printf("Tunjangan : %i\n", tunjangan);

    printf("Gaji bersih : %i\n", gajibersih);

    return 0;

}

Pembahasan

Ada bagian yang aneh pada soal, "gunakan array dan petualangan". Array adalah bentuk variabel dengan alokasi memory untuk banyak nilai. Yang saya bingung adalah istilah "petualangan" disini. Setahu saya tidak ada istilah "petualangan" dalam bahasa pemrograman, mohon dikoreksi jika memang terjadi typo.

Pada solusi pemrograman yang dibuat, kita memanfaatkan golongan untuk menjadi index dari Array.

Pada potongan code berikut golongan = golongan - 'A';

Berfungsi untuk mengubah karakter A menjadi nilai 0. Perlu diketahui karakter adalah sebuah huruf yang memiliki nilai angka pada komputer, atau ASCII. Sehingga bisa dioperasikan menggunkana operator aritmatika seperti penjumlahan pengurangan, dan lainnya.

Maka jika golongan adalah A dan dikurangi 'A' maka hasilnya adalah 0

jika golongan adalah B dan dikurangi 'A' maka hasilnya adalah 1

jika golongan adalah C dan dikurangi 'A' maka hasilnya adalah 2

Nilai ini bisa kita gunakan sebagai index dalam array gaji yang sudah didefinisikan sebelumnya.

Sehingga kita bisa akses secara dinamis array gaji menggunakan golongan sebagi indexnya.

Status kawin 2 anak (asumsi januari sampai desember 2016 bekerja penuh). Jumlah penghasilan neto Gunawan, S.H per bulan tahun 2016 adalah ...


Jumlah penghasilan neto selama sebulan untuk periode 2016 milik Gunawan S.H. yang merupakan seorang PNS Golongan III/B yang berstatus kawin memiliki 2 anak adalah sebesar Rp5.648.750.

Penjelasan dengan langkah-langkah:

Penghasilan Neto adalaha penghasilan yang menjadi dasar perhitungan pajak seseorang, dimana penghasilan neto didapat dengan cara mengurangi penghasilan bruto atau kotor dengan beberapa biaya biaya yang ada.

Diketahui : Gunawan S.H. seorang PNS golongan III/B yang berstatus sudah kawin dan memiliki 2 anak, pada periode Februari 2016 memiliki gaji pokok Rp3.500.000 dengan ketentuan tunjangan istri 10%, tunjangan anak 2% per anak, tunjangan fungsional Rp2.000.000, iuran pensiun 4, 75%.

Ditanya : Jumlah penghasilan neto Gunawan S.H. per bulan pada tahun 2016?

Jawab :

Gaji Pokok = Rp3.500.000

Tunjangan Istri = 10% × Rp3.500.000 = Rp350.000

Tunjangan anak = 2% × Rp3.500.000 = Rp70.000 untuk 1 anak

Jumlah gaji dan tunjangan keluarga = Rp3.990.000  

Tunjangan fungsional = Rp2.000.000

Jumlah penghasilan bruto = Rp5.990.000

Pengurangan  

1. Biaya Jabatan = 5% × Rp3.500.000 = Rp175.000

2. Iuran pensiun = 4,75% ×Rp3.500.000 = Rp166,250

Jumlah pengurangan perbulan = Rp341.250

Penghasilan Neto sebulan

= Rp5.990.000 - Rp341.250

= Rp5.648.750
