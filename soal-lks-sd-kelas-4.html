35. tolong di jawab soal kelas 6 lks


penyelesaian :

1) volume = π x r² x t

                = 3,14 x 35² x 40

                = 3,14 x 1225 x 40

                = 153.860 cm³

2) volume = luas alas x t

   172.480 = luas alas x 70

   luas alas = 172.480 : 70

                  = 2.464 cm²

3) volume = luas alas x t

   100.480 = 1.256 x t

                t = 100.480 : 1.256

                  = 80 cm

4) diameter = 0,84 m

   jari-jari = diameter : 2 = 0,84 : 2 = 0,42 m

   volume = π x r² x t

                 = 22/7 x 0,42² x 0,96

                 = 22/7 x 0,1764 x 0,96

                 = 3,725568 : 7

                 = 0,532224 m³
