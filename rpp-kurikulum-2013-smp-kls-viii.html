Sifat-Sifat Bilangan Berpangkat
a. Sifat Perkalian Bilangan Berpangkat
Jika m, n adalah bilangan bulat positif dan a bilangan bulat maka am x an = am + n.
Contoh Soal :
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa 36 x 33 = 39!
Jawab:
36 x 33 = ( 3 x 3 x 3 x 3 x 3 x 3) x (3 x 3 x 3)
6 faktor 3 faktor
= (3 x 3 x 3 x 3 x 3 x 3 x 3 x 3 x 3)
(6 + 3) faktor
= 26 + 3
= 29

b. Sifat Pembagian Bilangan Berpangkat
Jika m, n bilangan bulat positif dan m > n dengan a bilangan bulat dan a bukan 0, maka am : an = am - n.
Contoh Soal :
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa 36 : 33 = 33!
Jawab :
36 : 33 = 36
33
= (3 x 3 x 3 x 3 x 3 x 3)
3 x 3 x 3
= (3 x 3 x 3)
(6 - 3) faktor
= 36 - 3
= 33

c. Sifat Perpangkatan Bilangan Berpangkat
Jika a, m, dan n bilangan bulat maka (am)n = am x n.
Contoh Soal:
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa (43)2 = 46.
Jawab:
(43)2 = (43) x (43)
= (4 x 4 x 4) x (4 x 4 x 4)
3 faktor 3 faktor
= 4 x 4 x 4 x 4 x 4 x 4
(3 x 2) faktor
= 43 x 2 = 46

d. Sifat Perpangkatan pada Perkalian
Jika a, b bilangan bulat dan m bilangan bulat positifmaka (a x b)m = am x bm.
Contoh Soal=
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa (32)3 = 33 x 23.
Jawab:
(32)3 = (3 x 2) x (3 x 2) x (3 x 2)
= (3 x 3 x 3) x (2 x 2 x 2)
3 faktor 3 faktor
= 33 x 23

e. Sifat Perpangkatan pada Pembagian.
Jika a, b bilangan bulat (b bukan 0) dan m bilangan bulat positif, maka (a : b)m = am : bm.
Contoh soal:
Dengan cara menuliskan dalam bentuk pembagian, tunjukkan bahwa (5 : 3)4 = 54 : 34.
Jawab:
(5 : 3)4 = (5/3)4
= (5/3) x (5/3) x (5/3) X (5/3)
= (5 x 5 x 5 x 5) = 54 = 54 : 53
(3 x 3 x 3 x 3) 34

PEMBAHASAN

Hukum Hooke menyatakan bahwa pertambahan panjang suatu pegas berbanding lurus dengan gaya yang bekerja pada pegas tersebut.

F = k . x

dimana:

F = Gaya Pegas ( N )

k = Konstanta Pegas ( N/m )

x = Pertambahan Panjang Pegas ( m )

Periode dari pegas yang digetarkan bisa dirumuskan sebagai :

T = 2π √ (m/k)

dimana :

T = periode pegas ( s )

m = massa beban yang digantung ( kg )

k = konstanta pegas ( N/m )

Energi potensial pegas bisa di rumuskan dengan rumus berikut ini :

Ep = (1/2) k . x²

dimana :

Ep = Energi potensial pegas ( Joule )

k = konstanta pegas ( N/m )

x = pertambahan panjang pegas ( m² )

Mari kita sekarang kita coba menyelesaikan soalnya.

Soal No. 4 :

Amplitudo atau Simpangan Maksimum sesuai definisi di ukur dari titik seimbang ( titik O ) ke titik paling atas ( titik A ) atau ke titik paling bawah ( titik B ) , maka :

AO = BO = ½ AB = ½ (12) = 6 cm

Soal No. 5 :

Diketahui :

Lama Getaran = t = 1 menit = 60 sekon

Banyak Getaran n = 40 kali

Ditanyakan :

Periode = T = ?

Frekuensi = f = ?

Penyelesaian :

Untuk mencari Periode dan Frekuensi bisa di pakai rumus berikut ini :

T = t / n

T = 60 / 40

T = 3/2 sekon

f = n / t

f = 40 / 60

f = 2/3 Hz

Soal No. 6 :

Diketahui :

Lama Getaran = t = 1 menit = 60 sekon

Frekuensi Getaran = f = 5 Hz

Ditanyakan :

Banyak Getaran = n = ?

Penyelesaian :

Untuk mencari banyaknya getaran bisa di pakai rumus berikut ini :

f = n / t

5 = n / 60

n = 60 x 5

n = 300 kali

Soal No. 7 :

Periode dan frekuensi pada pegas hanya bergantung pada massa dan konstanta pegas sesuai dengan rumus :

T = 2π √ (m/k)

dimana :

T = periode pegas ( s )

m = massa beban yang digantung ( kg )

k = konstanta pegas ( N/m )

Periode dan frekuensi pada ayunan ( bandul ) hanya bergantung pada panjang bandul dan percepatan gravitasi sesuai dengan rumus :

T = 2π √ (L/g)

dimana :

T = periode bandul ( s )

L = panjang bandul ( m )

g = percepatan gravitasi ( m/s² )

Kesimpulan :

Amplitudo tidak berpengaruh pada frekuensi

19. Urutan teks pelangi kurikulum 2013 smp


Pelangi merupakan suatu busur spektrum besar yang terjadi karena pembiasan cahaya matahari oleh butir-butir air. Pelangi juga dianggap sebagai gejala optik dan meteorologi berupa cahaya beraneka warna saling sejajar yang tampak di langit atau medium lainnya. Pelangi tampak sebagai busur cahaya dengan ujungnya mengarah pada horizon pada suatu saat hujan ringan. Pelangi juga dapat dilihat di sekitar air terjun yang jelas. Deretan Penjelas : Pelangi dan efek cahaya lain di langit di sebabkan oleh cahaya yang membias dan menyimpang menjauhi partikel. Saat matahari terbenam, langit menjadi merah karena karena sinar matahari lewat melalui atmosfer yang jauh lebih tebal daripada ketika matahari berada tinggi di langit pada siang hari. Pelangi tidak akan tampak pada malam hari atau ketika cuaca mendung. Hal itu terjadi karena pelangi merupakan hasil dari pembiasan cahaya. Pada mulanya cahaya matahari melewati sebuah tetes hujan, kemudian di belokkan atau dibiaskan menuju tengah tetes hujan sehingga memisahkan cahaya putih menjadi warna spektrum. Interpretasi : Pelangi hanya dapat di lihat saat hujan bersamaan dengan matahari bersinar tapi dari sisi yang berlawanan dengan si pengamat. Posisi si pengamat harus berada di antara matahari dan terkesan air dengan matahari di belakang orang tersebut. Matahari, mata si pengamat dan pusat busur pelangi harus berada dalam satu garis lurus. Cahaya dengan panjang gelombang terpendek seperti ungu, terdapat di bagian kurva dan yang memiliki panjang gelombang terpanjang seperti merah terdapat pada bagian luar.

Penjelasan:

RENCANA PELAKSANAAN PEMBELAJARAN

(RPP)

Materi Pokok : Biologi

Kelas/semester : XI / 2

Materi Pokok : Sistem Regulasi Manusia

Sub Materi : Struktur dan fungsi dari bagian-bagian alat indera

Alokasi waktu : 2 x 30 Menit

KOMPETENSI INTI

Menjelaskan struktur dan fungsi organ manusia dan hewan tertentu, kelainan dan atau penyakit yang mungkin terjadi serta implikasinya pada salingtemas.

KOMPETENSI DASAR

Menjelaskan keterkaitan antara struktur, fungsi dan proses serta kelainan/penyakit yang dapat terjadi pada sistem regulasi manusia (saraf, endokrin, dan penginderaan).

INDIKATOR

Menyebutkan daerah pengecapan rasa pada lidah

Menyebutkan bagian yang menyusun indra pembau.

Menghubungkan fungsi indra pembau dengan indra pengecap

Membuat kesimpulan hasil praktikum

TUJUAN PEMBELAJARAN

Siswa dapat menyebutkan daerah pengecapan rasa pada lidah setelah berdiskusi dengan teman kelompoknya

Siswa dapat menyebutkan bagian yang menyusun indra pembau setelah berdiskusi dengan teman kelompoknya

Siswa dapat menghubungkan fungsi indra pembau dengan indra pengecap setelah melakukan praktikum indra pembau dan pengecap.

Siswa dapat membuat kesimpulan setelah melakukan praktikum indera pembau dan pengecap.

MATERI POKOK

Daerah pengecapan macam-macam rasa pada lidah manusia:

Rasa pahit : bagian pangkal lidah

Rasa asam : bagian pinggir lidah

Rasa manis : bagian ujung depan lidah

Rasa asin : bagian ujung lidah samping

Struktur indera pembau terdiri dari sel penyokong yang berupa sel epitel dan sel pembau yang berupa neuron sebagai reseptor.

Reseptor pembau dan pengecap saling berhubungan dan bekerja sama. Indra pembau atau pencium menerima stimulus berupa gas, sedangkan indra pengecap menerima stimulus berupa cairan.

METODE DAN MODEL PEMBELAJARAN

Metode : Praktikum, diskusi, tanya jawab

Model Pembelajaran : Inkuiri

Pendekatan : Saintifik

32. Analisis rpp kurikulum karakter dan kurikulum merdeka. Carilah perbedaan dan persamaan yang terdapat dalam rpp kurikulum karakter dan RPP kurikulum merdeka


Penjelasan:

Perbedaan

1. Fokus:

- Kurikulum Karakter: Kurikulum karakter bertujuan untuk mengembangkan karakter positif dalam siswa, seperti moral, etika, kejujuran, tanggung jawab, disiplin, kerja keras, dan sikap positif lainnya.

- Kurikulum Merdeka: Kurikulum Merdeka memiliki fokus pada pengembangan kemandirian dan pemikiran kritis siswa, mengajarkan keterampilan hidup, seperti berpikir, bertanya, berkomunikasi, berkolaborasi, dan mengelola diri sendiri.

2. Pendekatan:

- Kurikulum Karakter: Kurikulum karakter sering kali mengintegrasikan nilai-nilai karakter ke dalam semua mata pelajaran dan kegiatan sekolah. Pendekatan yang digunakan melibatkan pengajaran langsung dengan menggunakan metode dan teknik seperti cerita, permainan peran, diskusi, dan penilaian karakter.

- Kurikulum Merdeka: Kurikulum Merdeka mendorong pendekatan pembelajaran yang aktif dan kolaboratif. Siswa diberikan kebebasan untuk mengkaji materi secara mandiri dan memilih cara belajar yang paling sesuai untuk mereka. Guru bertindak sebagai fasilitator yang membantu siswa mengembangkan keterampilan pemecahan masalah dan pemikiran kritis.

3. Tujuan:

- Kurikulum Karakter: Tujuan dari Kurikulum Karakter adalah mengembangkan nilai-nilai karakter yang kuat pada siswa, sehingga mereka menjadi pribadi yang bertanggung jawab, berintegritas, dan memiliki empati terhadap orang lain.

- Kurikulum Merdeka: Tujuan dari Kurikulum Merdeka adalah mengembangkan siswa yang mandiri, aktif, dan kreatif, yang mampu mengambil tanggung jawab atas pembelajaran mereka sendiri. Mereka diharapkan dapat menghadapi tantangan dunia nyata dengan pemikiran kritis dan kemampuan beradaptasi.

Persamaan antara RPP Kurikulum Karakter dan RPP Kurikulum Merdeka adalah:

1. Menekankan pengembangan keterampilan hidup siswa.

2. Mengajarkan nilai-nilai positif dan moralitas.

3. Memprioritaskan pengembangan kepribadian siswa.

4. Mengikuti kurikulum yang ditetapkan secara nasional atau oleh lembaga pendidikan terkait.
